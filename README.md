# CV — Juan Gaitán-Villamizar

Live at [0xjgv.github.io/cv](https://0xjgv.github.io/cv/)

## Generate PDF

1. Open `index.html` in your browser (or visit the live URL)
2. Press `Cmd + P` (macOS) or `Ctrl + P` (Windows/Linux)
3. Set destination to **Save as PDF**
4. Use these settings for best results:
   - Paper size: **A4**
   - Margins: **None**
   - Background graphics: **enabled**
5. Save

Alternatively, from the command line using Chrome:

```sh
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
  --headless --print-to-pdf=cv.pdf --no-pdf-header-footer \
  --print-to-pdf-no-header index.html
```
